go-jsbox-skeleton
=================

|travis|_

Skeleton for a vumi javascript sandbox app. Use this as a starting point for writing a sandbox app using the `vumi-jssandbox-toolkit`_.

::

    $ npm install
    $ npm test

Take a look at the `docs`_ for more examples and the toolkit's api.
